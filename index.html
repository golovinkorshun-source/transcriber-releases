Установка

  1. Нажмите «Скачать для Windows» и запустите Transcriber-Setup.exe.
  2. Windows покажет синее окно «Система Windows защитила ваш компьютер» — это SmartScreen: приложение новое и без платной подписи, для маленьких программ это обычное дело. Нажмите «Подробнее», затем «Выполнить в любом случае».
    Если кнопки «Выполнить в любом случае» нет — сначала кликните «Подробнее», она появится под текстом.
  3. Пройдите установщик до конца и запустите Транскрибатор. При первой диктовке Windows сам спросит разрешение на микрофон — разрешите.

Как диктовать

Откройте вкладку Настройки → «Горячая клавиша», выберите режим «Одна клавиша», нажмите «Записать клавишу» и нажмите правый Ctrl. Он под правой рукой и его почти никто не использует, так что случайных нажатий не будет.

Дальше в любом приложении: поставьте курсор куда нужно, зажмите правый Ctrl, говорите, отпустите. Через секунду текст вставится сам — со знаками препинания.

Если не работает

  • Текст не появляется вовсе
    Проверьте интернет: распознавание идёт на сервере. Откройте само окно Транскрибатора — статус сервера виден на главном экране. Если там ошибка, напишите мне.
  • Микрофон молчит
    Параметры Windows → Конфиденциальность → Микрофон: доступ приложениям включён, Транскрибатору — разрешён. Затем перезапустите приложение.
  • Слышит, но вставляет не туда или не вставляет
    Кликните в поле, куда диктуете, чтобы там мигал курсор, и попробуйте снова. Текст последней диктовки всегда лежит в окне приложения на вкладке «История» — оттуда его можно скопировать.
  • Ничего не помогло
    Напишите мне и приложите скриншот окна приложения — разберёмся.

Приватность — честно

Пока вы удерживаете клавишу, звук с микрофона уходит на мой сервер, а оттуда — в OpenAI для распознавания. Мой сервер аудио не сохраняет вовсе; OpenAI обрабатывает его по правилам своего API — не использует для обучения моделей. Обратно приходит только текст. История распознанного хранится локально у вас и вычищается по настройке.

Вне зажатой клавиши приложение ничего никуда не отправляет. Если такой расклад не подходит — просто не пользуйтесь, я пойму.
