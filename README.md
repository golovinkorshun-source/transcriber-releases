# Транскрибатор — диктовка для Windows

Зажал клавишу — сказал — отпустил: текст с пунктуацией уже вставлен туда,
где стоял курсор. Русский и английский.

**Сайт с инструкцией:** https://golovinkorshun-source.github.io/transcriber-releases/

**Скачать установщик:**
[Transcriber-Setup.exe](https://github.com/golovinkorshun-source/transcriber-releases/releases/latest/download/Transcriber-Setup.exe)
(Windows 10/11, 64-бит)

## Установка

1. Скачайте и запустите `Transcriber-Setup.exe`.
2. Windows покажет синее окно **«Система Windows защитила ваш компьютер»** —
   это SmartScreen: приложение новое и без платной подписи, для маленьких
   программ это обычное дело. Нажмите **«Подробнее»**, затем
   **«Выполнить в любом случае»** (кнопка появляется только после
   «Подробнее»).
3. Пройдите установщик, запустите Транскрибатор. При первой диктовке
   Windows сам спросит разрешение на **микрофон** — разрешите.

## Как диктовать

Вкладка **Настройки → «Горячая клавиша»** → режим **«Одна клавиша»** →
**«Записать клавишу»** → нажмите **правый Ctrl**. Он под правой рукой и
почти никем не используется — случайных нажатий не будет.

Дальше в любом приложении: курсор в нужное поле, зажмите правый Ctrl,
говорите, отпустите — через секунду текст вставится сам.

## Если не работает

- **Текст не появляется** — проверьте интернет: распознавание идёт на
  сервере. Статус сервера виден на главном экране приложения.
- **Микрофон молчит** — Параметры Windows → Конфиденциальность → Микрофон:
  доступ включён и разрешён Транскрибатору. Перезапустите приложение.
- **Вставляет не туда** — кликните в поле, чтобы там мигал курсор, и
  повторите. Текст последней диктовки всегда есть во вкладке «История».
- **Ничего не помогло** — напишите мне в Telegram со скриншотом окна.

## Приватность — честно

Пока вы удерживаете клавишу, звук с микрофона уходит на сервер владельца
проекта, а оттуда — в OpenAI для распознавания. Сервер аудио не сохраняет;
OpenAI обрабатывает его по правилам своего API (не использует для обучения
моделей). Обратно приходит только текст. Вне зажатой клавиши приложение
ничего никуда не отправляет.
